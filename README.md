# WheatSong · 开源项目集

欢迎来到 **WheatSong** 的开源仓库！这里汇集了「面包奇旅」系列游戏及其周边工具的全部前端源代码。所有项目均使用原生 HTML/CSS/JavaScript 构建，并基于 **Supabase** 提供后端服务，完全开源，可供学习、二次开发或自部署。

# 🎮 游戏部分（Games）

「面包奇旅」是一款以「废弃酵母」大陆为舞台的奇幻探索游戏，包含完整的剧情、角色成长、任务系统与活动玩法。整个系列由以下 6 个独立站点组成，它们共同构成一个完整的游戏生态。

---

## 1. 🚪 登录认证中心（Login Gateway）

**域名**：`wheatsong-breadimpact-login.netlify.app`

**功能**：
- 用户账号密码登录（支持自动登录）
- 隐私政策确认与跳转
- 注册入口引导
- 3D 场景品牌动画（降级兼容）
- 登录成功后携带 `user_id` 跳转至主游戏

**技术亮点**：
- 使用 Supabase Auth 进行身份验证
- 页面刷新时强制校验来源与 URL 参数，防止未授权访问
- 自动登录基于 localStorage 缓存

---

## 2. 🌾 主游戏世界（Main Game）

**域名**：`wheatsong-breadimpact.netlify.app`

**功能**：
- 2D 横版探索场景（动态天空、森林、城镇、建筑）
- 摇杆控制角色移动（基于 nipplejs）
- 剧情对话系统（逐字显示、点击推进）
- 任务系统（主线任务自动触发、进度追踪、奖励发放）
- 背包系统（物品分类、详情查看）
- 活动入口（跳转至战斗子游戏）
- 「酵醒等阶」经验成长系统（带升级动画）
- 实时数据云端同步（位置、任务、物品、经验）

**技术亮点**：
- 纯 JavaScript 游戏引擎（无第三方框架）
- 自定义碰撞与视差滚动
- 任务进度以字符串编码（如 `0_1_1_1_0`）便于存储
- 与 Supabase 实时同步（每 200ms 保存位置）

---

## 3. 🌐 品牌形象官网（Official Site）

**域名**：`wheatsong.netlify.app`

**功能**：
- 世界观展示（「三月神话」「废弃酵母」「赤月王朝」）
- 角色图鉴（面包、八萬、神秘人，含模态框详情）
- 道具展示（黄金酵母、麦粒、丝织之羽等）
- 章节时间线（可展开查看剧情概要）
- 交互式三月选择器（点击唤醒月之传说）
- 引导至登录页的 CTA 按钮

**技术亮点**：
- 纯 CSS 动画（浮动光点、鼠标光晕、滚动触发）
- Canvas 气泡背景
- 角色模态框与响应式设计

---

## 4. 📝 独立注册页（Signup Page）

**域名**：`wheatsong-breadimpact-signup.netlify.app`

**功能**：
- 新用户注册（用户名、密码、邮箱）
- 与 Supabase Auth 对接
- 注册成功后引导至登录页

**说明**：此页面与登录页分离，降低认证模块耦合度，便于独立维护。

---

## 5. ⚔️ 战斗/活动子游戏（Battle Sub-game）

**域名**：`wheatsong-breadimpact-battle.netlify.app`

**功能**：
- 承载「无极限超激格斗赛」等限时活动
- 接收主游戏传递的 `user_id` 及玩家状态
- 独立于主游戏运行，可通过主游戏内活动按钮打开

**技术栈**：与主游戏共享 Supabase 数据，未来可扩展为独立小游戏平台。

---

## 6. 📄 隐私政策页（Privacy Policy）

**域名**：`wheatsong-privacy.netlify.app`

**功能**：
- 展示用户隐私协议与条款
- 被登录页和官网共同引用，满足法律合规要求

# 🛠️ 工具部分（Tools）

## 📦 代码储存器（Code Storage）

**项目简介**：  
一款面向程序员的云端代码片段管理工具，支持纯文本与多种编程语言代码的存储、编辑、搜索与下载。所有数据保存在 Supabase 数据库中，实现跨设备访问。

**核心功能**：
- **新建文档**：支持纯文本（TXT）和代码（CODE）两种类型
- **代码类型后缀选择**：支持 `.html`、`.css`、`.js`、`.py`、`.java` 等 20+ 种常见扩展名
- **实时统计**：行数、中文字数、总字符数
- **文本查找**：高亮所有匹配项，支持上下导航，当前匹配项高亮显示
- **保存/清空/下载/删除**：完整 CRUD 操作
- **文件详情**：查看文件名、格式、行数、字数、创建/修改时间等元信息
- **快捷键**：`Ctrl+S` 快速保存，`Esc` 取消高亮

**技术亮点**：
- 基于 `contenteditable` 实现富文本编辑（纯文本模式）
- 自定义高亮引擎，支持正则匹配与滚动定位
- 光标位置保存与恢复，保证高亮后编辑不丢失焦点
- 与 Supabase 实时同步，数据持久化

**部署方式**：可直接在 Netlify 或任何静态托管服务上部署，仅需配置 Supabase 表结构和 RLS 策略（详见项目内文档）。

# 🔧 通用技术栈

- **前端**：原生 HTML5 + CSS3 + JavaScript（ES6+）
- **后端**：Supabase（PostgreSQL 数据库 + Auth + Storage）
- **托管**：Netlify（静态站点）
- **依赖库**：
  - Supabase JS SDK（数据库与认证）
  - Three.js（登录页 3D 场景）
  - Nipple.js（主游戏摇杆控制）
  - VConsole（调试工具）

# 🚀 快速开始

每个子项目均为独立静态页面，可直接打开 `index.html` 运行。若需连接 Supabase，请确保：

1. 创建 Supabase 项目，并执行对应的建表语句（详见各项目说明）。
2. 将 `SUPABASE_URL` 和 `SUPABASE_ANON_KEY` 替换为你的实际值。
3. 配置 Netlify 重定向规则（如 SPA 路由）或直接部署为单页。
